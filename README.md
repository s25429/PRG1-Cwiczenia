# Ćwiczenia C++
> Tutaj znajdują się pliki c++ zadane do zrobienia z ćwiczeń programowania.

## Info ogólne
- Zadania posiadają notkę commitu, gdzie jest napisana data zajęć ćwiczeniowych, na których było zadane zrobienie zadań.
- Pliki z wykładów, lub inne projekty są w innych repozytoriach

## Pliki i katalogi
- W /src są pliki source .cpp
- W /build są pliki wynikowe po kompilacji .bin
- Makefile jest używany do kompilowania plików .cpp do .bin

## Kompilacja
- Dla pliku 1.0.0.1-hello-world.cpp, który znajduje się w /src
- `make build/1.0.0.1-hello-world.bin`
- Plik 1.0.0.1-hello-world.bin pojawia się w katalogu /build
- Aby wyczyścić katalog /build używa się komendy `make clean`
